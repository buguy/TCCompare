General 3
 

 Single Display - Full screen mode with 2 PCs

Use same OS
Setup: 
a) One DUT connects to 2 PCs through different input port.
b) All PCs connected to same network.
c) PC2 has to go to DDPM app > KVM > Network KVM and click on it. 

Test Steps:
 
1. Open Software > KVM > Full screen > proceed until completion.
 
2. Check the KVM setup displayed and KVM settings.
 
3. Able to copy file to another PC.
 
4. Snap an image on the clipboard or Screenshot of a picture in PC.
 
5. Unplug video cable from PC1 and plug in cable again.
 
6. Unplug video cable from PC2 and plug in cable again.

7. In software > KVM > Right click the tool button > Select 'Switch PC‘.
 
8. Use Network KVM: Switch between PC's hotkey to switch video.
 
9. Use "Restore the mouse cursor to a local computer." hotkey on:
a) PC1 Keyboard
b) PC2 Keyboard
c) External Keyboard connected to PC1
d) External Keyboard connected to PC2
e) With DUT screen showing PC1 screen.
External keyboard connected to DUT.
f) With DUT screen showing PC2 screen.
External keyboard connected to DUT.
 

Expected Result : 
 
2. Verify the following in all PCs:
  • Monitor icon
    • Single display monitor is displayed with which PC is showing. PC1 or PC2
  • PC info
    • PC1 and PC2 information showing as below:
      • Connection between PC1 and PC2 with DUT
      • IP address showing is correct
      • PC OS version is correct
  • KVM Settings
    • On all PC, make sure Auto connect is enabled. Go to KVM settings > Manage Cluster > Auto Connect is ticked
3. Verify the following:
  • Able Copy single or multiple file from PC1 to PC2.
  • Able Copy single or multiple file from PC2 to PC1.
  • File can be in any format.
4. Switch to different PC, able to paste the image from the source clipboard.
 
5. Able to auto resume network KVM.
 
6. Able to auto resume network KVM.
 
7. DUT screen switch to the right clicked tool button PC screen.
Example: If the right clicked tool button is PC2, then DUT is now displaying PC2 screen.
 
8. Able to toggle video input with hotkey correctly.
 
9. Verify the following:
a) & c) & e) Mouse cursor should be back to PC1's screen or monitor screen connected to PC1.
b) & d) & f) Mouse cursor should be back to PC2's screen or monitor screen connected to PC2.
